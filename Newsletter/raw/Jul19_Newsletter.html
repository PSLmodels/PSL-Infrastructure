Cost-of-Capital-Calculator

Evaluates the effect of US federal taxes on businesses' investment incentives

Releases and new features
  • Release 1.0.3
    • Improvements to the web application interface.
  • Release 1.0.2
    • Updates Specification() class to use ParamTools.
    • Adds compatibility with Tax-Calculator 2.4.

Tax-Brain

Tax-Brain is an integrator package for PSL tax models

Releases and new features
  • Release 2.3.0
    • Adds automated report capabilities (PR #69).
    • Adds Tax-Brain command line interface (PR #67 and PR #68).
    • Updates COMP table outputs for improved readability (PR #66).
    • Allows users to specify an alternative policy to use as the baseline, rather than current law (PR #64).

Tax-Calculator

Tax-Calculator is a USA federal individual income and payroll tax microsimulation model

Releases and new features
  • Release 2.4.0
    • Improves calculation of qualified business income deduction (PR #2345).
