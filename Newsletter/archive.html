DSGE.jl

The Julia-language implementation of the Federal Reserve Bank of New York's DSGE model

The DSGE.jl package, newly cataloged in the PSL, is the Julia-language implementation of the Federal Reserve Bank of New York's dynamic stochastic general equilibrium (DSGE) model. It provides general code to estimate many user-specified DSGE models.

Check out this introductory post and the package's documentation for more details.

OG-USA

An overlapping-generations model for evaluating fiscal policy in the United States

Releases and new features

  • Started the transition to a country-agnostic overlapping generations modeling platform, OG-Core.
  • The first step of the transition, modifying the code base, is underway (#725).

OpenFisca-UK

A model of the United Kingdom's tax system based on the OpenFisca platform.

Releases and new features

  • Developed a new package, openfisca-uk-data, for handling datasets used for this model.
  • Changes to ensure that the model benefits from updates from OpenFisca.
  • Speed improvement by converting all variables to yearly.

PCI-China

A machine learning algorithm that predicts China's policy changes by mining its state-controlled media

General updates

  • Weifeng Zhong discussed the PCI projects and how Chinese propaganda works in the Shootin' Straight with Ken Buck podcast.
  • Weifeng Zhong discussed the PCI projects and the revival of Maoism in China in the History As It Happens podcast.

TaxData

A library that prepares microdata files to be used by Tax-Calculator

Releases and new features

  • Release 0.3.0    Switched the stage 2 linear programming solver to Tulip (#393) and updated to the February 2021 CBO economic projections (#394).
  • Releases 0.3.1 and 0.3.2 with bug fixes.

Tax-Calculator

A USA federal individual income and payroll tax microsimulation model

Releases and new features

  • Release 3.2.0    (1) Updated the current law baseline to include the CARES Act, the Consolidated Appropriations Act of 2021, the American Rescue Plan Act, the Consolidated Appropriations Act of 2021, as well as updates to EITC parameters; (2) Updated growfactors and weights for the PUF and CPS data to reflect updates to TaxData; (3) Other features and bug fixes (#2604).

General updates

  • City of Philadelphia created tool to use American Community Survey with Tax-Calculator.
